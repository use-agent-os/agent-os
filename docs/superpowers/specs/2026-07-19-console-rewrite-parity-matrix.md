> **MCP surface added - 2026-07-22.** Upstream `main` introduced managed MCP
> server configuration, live connection control and OAuth authorization in the
> legacy console. The React console now includes the matching `/mcp` and
> `/mcp/oauth/callback` routes, all six `mcp.*` RPC calls, configuration CRUD,
> responsive editor and the Robinhood integration inside the unified Control
> design system.

> **Agent setup consolidated - 2026-07-23.** The React console now exposes one
> **Agent setup** navigation item and a canonical `/settings` workspace.
> `/setup` and `/config` remain compatibility routes and open Guided and
> Advanced mode respectively. The workspace reads a coherent, redacted
> `config.snapshot`; Guided mode keeps the specialized onboarding mutations,
> while Advanced mode retains `config.patch` / `config.apply` and covers the
> complete current `GatewayConfig` surface. Both editors share revision-aware,
> persist-first writes and fail closed when the active file diverges from the
> running gateway.

> **Plan 2 complete — 2026-07-20.** All 11 standard views (approvals, logs,
> overview, channels, agents, sessions, usage, config, skills, cron, setup) +
> the approval-monitor service migrated and reviewed; a 12-surface batch parity
> audit (76 agents) found 21 deltas, all fixed via the parity-fix-round workflow
> (one verify pass caught a fake in-flight-guard test → remediated with a
> mutation-verified rewrite). Gates: FE `npm run check` green (829 tests / 31
> files; `vite build` clean, ~200kB gz JS), Python green (1559 gateway/parity),
> legacy UI byte-identical. **13/13 routes are real views — `chat` migrated in
> Plan 3 (2026-07-21), the StubView is retired.** Two `pending` rows remain — both cutover-plan items (tokenViz
> flag, custom base_path). Owner decisions honored where legacy behavior was
> at stake (T6 dirty-guard, T11 cron editor, T12 secrets). Follow-ups deferred:
> ConfirmDialog unification (structural CSS divergence; ModalShell drift already
> closed), live-browser smoke of all views (folds into cutover).


> **Plan 1 complete — 2026-07-20.** Layer-0/1 foundation (bootstrap, WS-RPC,
